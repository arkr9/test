# 一、整体安排思路

北京核心景点大致可以分成几片：

1. **中轴线核心区**  
   天安门、故宫、景山公园、北海公园  
   ——适合安排在同一天，距离近，历史文化浓度高。

2. **南城传统文化区**  
   天坛、大栅栏、前门  
   ——都在北京老城南边，交通方便，可以串起来玩。

3. **西城胡同/夜游休闲区**  
   什刹海、南锣鼓巷、鼓楼/钟楼、恭王府  
   ——更适合慢逛、拍照、吃饭、夜景。

因此，建议安排为：

- **Day 1：天安门 + 故宫 + 景山 + 什刹海夜景**
- **Day 2：天坛 + 前门/大栅栏 + 可加国家博物馆或天安门夜景**
- **Day 3：恭王府 + 南锣鼓巷 + 什刹海深度游 / 北海公园**

这样行程比较合理，既有历史地标，也有胡同氛围和夜晚休闲。

# 二、详细3日行程安排

---

## Day 1：天安门—故宫—景山公园—什刹海

**关键词：北京中轴线、皇城核心、经典首日必打卡**

### 1. 上午：天安门广场 + 天安门外观

**建议游览时间：1~1.5小时**

- 早上尽量早点到，避免排队和人流高峰。
- 可以先去**天安门广场**，感受北京的城市象征。
- 周边还能看到：
  - 人民英雄纪念碑
  - 人民大会堂（外观）
  - 国家博物馆（外观，若有兴趣可另安排）

**建议：**

- 天安门区域安检较严格，尽量轻装出行。
- 身份证一定带好。
- 如果你特别想看**升旗仪式**，需要更早出发，但会非常早，适合体力好且愿意早起的人。

---

### 2. 上午到下午：故宫博物院

**建议游览时间：3.5~5小时**

从天安门进入故宫，路线非常顺。

**推荐游览路线：**
午门 → 太和门 → 太和殿 → 中和殿 → 保和殿 → 乾清宫 → 交泰殿 → 坤宁宫 → 御花园 → 神武门出

如果时间比较充裕，可重点看- 珍宝馆

- 钟表馆
- 东西六宫部分区域

**为什么这么安排：**

- 故宫是北京最核心的景点之一，第一次来一定要留足时间。
- 走完整体中轴线后，再挑重点展馆，不容易太累。

**小提示：**

- **故宫门票需要提前预约**，尤其节假日一定要提前。
- 建议准备一点水和小零食，但不要带太多，走路会很累。
- 可以租讲解器，或者提前准备电子讲解，体验会好很多。

---

### 3. 下午：景山公园

**建议游览时间：1~1.5小时**

从故宫**神武门**出来，对面就是**景山公园**，非常适合顺路安排。

**亮点：**

- 登上万春亭，可以俯瞰故宫全景。
- 这里是拍故宫全貌最经典的位置之一。
- 故宫看完再来景山，会更有整体感。

如果天气好，拍照会非常出片。

---

### 4. 傍晚到晚上：什刹海

**建议游览时间：2~3小时**

从景山出来，可以打车或地铁前往**什刹海**。

**什刹海适合怎么玩：**

- 沿湖散步
- 看后海夜景
- 逛酒吧街
- 感受老北京胡同与现代夜生活结合的氛围

**可顺带打卡：**

- 烟袋斜街
- 鼓楼、钟楼（外观夜景）
- 银锭桥

**晚餐建议：**
可以在什刹海附近吃北京特色：

- 涮羊肉
- 炙子烤肉
- 北京菜
- 老北京炸酱面

---

## Day 2：天坛—前门—大栅栏—夜游前门

**关键词：老北京风貌、祭天文化、商业老街**

### 1. 上午：天坛公园

**建议游览时间：2.5~3小时**

天坛比较适合早上去，环境舒服，也能看到很多本地人在晨练。

**推荐路线：**
东门入园 或 南门入园 → 祈年殿 → 皇乾殿 → 丹陛桥 → 回音壁 → 圜丘

**核心看点：**

- **祈年殿**：天坛最标志性的建筑
- **回音壁**：很有趣
- **圜丘**：祭天重要场所

**为什么安排上午：**

- 天坛园区很大，早上气温相对舒服，拍照也好看。
- 天坛整体节奏没有故宫那么紧张，适合轻松一点地游览。

---

### 2. 中午：前门附近午餐

天坛玩完后，前往**前门/大栅栏**，路程不远。

**午餐建议区域：**

- 前门大街
- 大栅栏
- 鲜鱼口街

这里老字号比较集中，能体验北京老味道。

可尝试：

- 北京烤鸭
- 卤煮
- 豆汁（想体验地道北京味可试）
- 爆肚
- 门钉肉饼

---

### 3. 下午：大栅栏 + 前门大街

**建议游览时间：2~3小时**

这是老北京商业街区代表，和天坛的文化氛围能形成很好的搭配。

**可逛内容：**

- **大栅栏**：老字号云集，适合买点伴手礼
- **前门大街**：步行街，适合散步拍照
- **鲜鱼口街**：小吃比较集中
- **北京坊**：更时尚一点，拍照也不错

**推荐体验：**

- 看老建筑
- 逛老字号店铺
- 买北京特产
- 吃小吃

**适合购买的伴手礼：**

- 稻香村点心
- 果脯
- 茶叶
- 文创小物

---

### 4. 晚上：前门夜景 / 天安门夜景

**建议游览时间：1.5~2小时**

前门晚上灯光亮起后很有氛围，适合慢慢逛。  
如果你还想再感受一次北京地标夜色，可以从前门步行或乘车去看**天安门夜景**。

**这一天节奏相对轻松**，不会像故宫那天那么累。

---

## Day 3：恭王府—南锣鼓巷—北海公园/什刹海深度游

**关键词：王府文化、胡同漫步、休闲收尾**

这一天安排得更灵活一些，适合在前两天暴走后，稍微放慢节奏。

### 方案A：更经典更顺路

### 1. 上午：恭王府

**建议游览时间：2小时**

恭王府在什刹海附近，名气很高，是保存较完整的清代王府。

**亮点：**

- 王府建筑格局
- 花园设计精致
- 历史故事很多，值得听讲解

如果你对清代历史、建筑、园林有兴趣，这里很值得去。

---

### 2. 中午：什刹海/后海附近午餐

这里餐厅很多，环境也不错，适合慢慢吃一顿。

---

### 3. 下午：南锣鼓巷 + 胡同漫步

**建议游览时间：2~3小时**

**南锣鼓巷**是热门胡同街区，适合：

- 逛小店
- 买文创
- 吃小吃
- 拍胡同照片

不过这里游客比较多，如果你更喜欢安静一点，也可以从南锣鼓巷延伸到周边胡同随意走走，反而更有北京味道。

---

### 4. 傍晚：北海公园 或 再次什刹海休闲

**建议游览时间：1.5~2小时**

如果体力还可以，推荐去**北海公园**：

**亮点：**

- 皇家园林
- 白塔很经典
- 湖景漂亮
- 比较适合休闲散步

如果想更轻松一些，就继续在**什刹海**附近喝茶、看夜景、坐坐即可。

# 四、交通建议

北京旅游最方便的方式一般是：  
**地铁为主 + 打车补充**

### 适合地铁到达的景点

- 天安门
- 故宫周边
- 天坛
- 前门/大栅栏
- 南锣鼓巷
- 什刹海

### 打车更方便的情况

- 一天最后一站去吃饭或回酒店
- 景点之间不想换乘太多的时候
- 带老人小孩出行时

**建议：**

- 住在**前门、王府井、东单、西单、鼓楼附近**都比较方便。
- 如果3天都以市区景点为主，尽量住地铁沿线。

---

# 五、门票和预约提醒

以下景点建议提前看预约情况：

1. **故宫**  
   - 热门中的热门  
   - **必须尽早预约**

2. **天安门广场**  
   - 进入相关区域通常需要预约或核验，政策可能调整  
   - 出发前最好查看最新规定

3. **天坛**
   - 一般比较好买，但节假日建议提前安排

4. **恭王府**
   - 节假日也可能比较热门，建议提前订票
